Check out some of my projects:
Lumboard - The main purpose of Lumboard is to help users text more quickly and easily. It provides the comfort of spelling out words that are usually shortened, all with the tap of one key. The keyboard's design was inspired by Matias Duarte's Material Design. This application was built at the University of Waterloo at at Hack The North.

DSAC Telemetry Filter - This program is used to filter out data packets from the Deep Space Atomic Clock that fit into a user-selected range. It then prints out these packets to a spreadsheet. This was written as I was learning to code during my summer long internship as a part of the Jet Propulsion Laboratory's Summer High School Internship under NASA's MUREP.
