lumboard
the main purpose of lumboard is to help users text more quickly and easily. it provides the comfort of spelling out words that are most-frequently shortened, all with the tap of one key. the keyboard's design was inspired by matias duarte's material design language. this application was built at the university of waterloo at hack the north.


dsac telemetry filter
this program is used to filter out data packets from the deep space atomic clock (dsac), launching later this year. the user input is parsed to identify a time range and the packets in that range are converted into csv. i wrote this as i learned to code during my summer high school internship program at nasa's jet propulsion laboratory in pasadena.
